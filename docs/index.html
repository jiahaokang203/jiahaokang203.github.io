游戏远程启动器

不在电脑前，也能通过微信发送一条消息远程启动游戏。 每个游戏独立配置，支持自动按键、点击和图像识别动作链。

核心功能

微信远程启动

监听指定微信会话，发送「启动游戏名」即可触发。

多游戏独立配置

每个游戏单独保存 exe 路径与动作配置，切换更轻松。

动作引擎

支持 wait、key、hotkey、click、image_click。

本地测试先行

先本地验证动作，再远程触发，减少误操作风险。

三步上手

  1. 1配置游戏 exe 与动作 JSON，并保存。
  2. 2填写微信会话名称，启动监听。
  3. 3在微信发送「启动原神」等命令远程执行。

常见问题

必须安装什么环境？

Python 环境 + 项目依赖。若要微信远程功能，需要额外安装 wxauto 并登录电脑版微信。

image_click 为什么偶尔失败？

图像识别受分辨率和界面状态影响，建议在当前分辨率下重新截取模板。

可以用于 GitHub Pages 吗？

可以，本页面是纯静态站点，直接部署 `docs/` 目录即可。
